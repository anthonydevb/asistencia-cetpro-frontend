# Imágenes del Sistema

## Logo de la Empresa

Para agregar el logo de tu empresa en la página de login:

1. Coloca tu imagen del logo en esta carpeta con el nombre: `logo-empresa.png`
2. Formatos soportados: PNG, JPG, SVG
3. Tamaño recomendado: 400x400px o superior (para mejor calidad)
4. El logo debe tener fondo transparente para mejor visualización

Si no tienes una imagen, el sistema mostrará solo el nombre "CETPRE" y el subtítulo.
